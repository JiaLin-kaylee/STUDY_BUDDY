# STUDY_BUDDY

I am building an interactive soft robot designed to support students and educators with attention deficit (help with accountability when doing work). The goal is to create a companion that provides grounding cues, emotional comfort, and gentle interaction through movement, texture, and sound. The robot is inspired by USC’s Blossom project, but I am adapting the design to focus specifically on ADHD‑supportive behaviors such as soft tactile feedback.

This robot is meant to feel alive, approachable, and calming — not overstimulating. Every design decision, from the crochet body to the motion curves, is centered on that intention.
